# 导入导出

LtdView 支持多种导入导出格式，方便配置复用和离线展示。

## 导出 JSON 配置

导出完整的仪表盘配置为 JSON 文件。

### 包含内容

```json
{
  "id": "dashboard-uuid",
  "name": "销售数据大屏",
  "width": 1920,
  "height": 1080,
  "background": "#0a1929",
  "theme": "dark",
  "components": [
    {
      "id": "comp-uuid",
      "type": "bar-chart",
      "x": 100,
      "y": 50,
      "width": 600,
      "height": 400,
      "zIndex": 1,
      "props": { "title": "月度销售额", "theme": "dark" },
      "data": { "type": "static", "source": [...] },
      "interaction": { "click": { "type": "drillDown", ... } },
      "animation": { "entrance": "fadeIn", "duration": 500 },
      "style": { "background": "transparent", "borderRadius": 8 }
    }
  ]
}
```

### 导出方式

1. **顶部工具栏**：点击「导出」下拉菜单 → 导出 JSON 配置
2. **右侧属性面板**：画布属性 → 导入导出 → 导出 JSON 配置

## 导入 JSON 配置

导入之前导出的 JSON 配置文件，恢复仪表盘。

### 导入方式

1. **顶部工具栏**：点击「导入」按钮
2. **右侧属性面板**：画布属性 → 导入导出 → 导入 JSON 配置

### 注意事项

- 导入会覆盖当前画布的所有内容
- 组件 ID 会重新生成
- 数据源配置会保留

## 导出独立 HTML

导出一个自包含的 HTML 文件，无需服务器即可在浏览器中打开。

### 文件内容

- 完整的 HTML 结构
- 内嵌 CSS 样式
- ECharts CDN 引用
- 自适应缩放脚本

### 支持的组件

| 组件 | 导出效果 |
|------|----------|
| 图表 | ECharts 渲染（需要网络加载 CDN） |
| 表格 | 原生 HTML table |
| 表单 | 原生 HTML input/select |
| 标题/副标题 | HTML 文本 |
| 数字翻牌器 | 静态数字展示 |
| 图片 | HTML img 标签 |
| 分割线 | HTML hr/div |
| 装饰边框 | CSS border |

### 导出方式

1. **顶部工具栏**：点击「导出」下拉菜单 → 导出独立 HTML
2. **右侧属性面板**：画布属性 → 导入导出 → 导出独立 HTML

### 使用方式

双击导出的 HTML 文件即可在浏览器中打开，自动适配窗口大小。

## 大屏分辨率适配

### 预设分辨率

| 预设 | 分辨率 |
|------|--------|
| 720p | 1280 x 720 |
| 1080p | 1920 x 1080 |
| 2K | 2560 x 1440 |
| 4K | 3840 x 2160 |
| 平板 | 1024 x 768 |
| 手机 | 375 x 812 |

### 适配方式

- **编辑器**：画布区域支持 25%-200% 缩放
- **预览模式**：5 种自适应缩放模式
  - 适应窗口（contain）：完整显示，可能有留白
  - 填充窗口（cover）：铺满屏幕，可能裁切
  - 原始大小：1:1 显示
  - 宽度适配：宽度铺满
  - 高度适配：高度铺满
- **独立 HTML**：自动缩放脚本，窗口大小变化时重新计算
