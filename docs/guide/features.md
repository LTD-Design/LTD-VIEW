# 功能特性

## 编辑器

### 三栏布局

编辑器采用经典三栏布局：

- **左侧**：组件库面板，分组展示所有可用组件
- **中间**：画布区域，组件的可视化编辑区
- **右侧**：属性面板，选中组件后显示配置项

### 画布操作

| 操作 | 方式 |
|------|------|
| 拖拽组件 | 从左侧拖入画布 |
| 移动组件 | 在画布中拖拽 |
| 调整大小 | 选中后拖拽边角手柄 |
| 多选 | 框选 或 Ctrl+点击 |
| 删除 | Delete / Backspace 键 |
| 撤销 | Ctrl+Z |
| 重做 | Ctrl+Y / Ctrl+Shift+Z |
| 缩放 | 工具栏 +/- 按钮 |

### 组件层级

支持调整组件层级：
- 置顶 / 置底
- 上移一层 / 下移一层

## 图表组件

支持 9 种 ECharts 图表：

| 组件 | 说明 |
|------|------|
| 柱状图 | 分类数据对比 |
| 折线图 | 趋势数据展示 |
| 饼图 | 占比数据展示 |
| 仪表盘 | 单值指标展示 |
| 雷达图 | 多维度对比 |
| 散点图 | 变量关系展示 |
| 漏斗图 | 转化率展示 |
| 热力图 | 矩阵数据可视化 |
| 矩形树图 | 层级面积占比 |

## 文本与媒体组件

| 组件 | 说明 |
|------|------|
| 标题/副标题 | 文字展示 |
| KPI 卡片 | 核心指标卡片，带趋势箭头 |
| 进度条 | 水平/垂直进度条 |
| 图片 | 多种填充模式 |
| 视频 | HTML5 视频播放器 |
| 排名列表 | 带金银铜色的排名展示 |

## DataV 装饰组件

基于 @kjgl77/datav-vue3 封装，提供大屏视觉增强：

| 组件 | 说明 |
|------|------|
| 边框 | 13 种科技感边框样式 |
| 装饰器 | 10 种线条装饰样式 |
| 轮播表格 | 自动轮播数据表格 |
| 数字翻牌器 | 数字翻牌动画 |
| 水球图 | 水位百分比动画 |
| 加载动画 | 科技感加载效果 |

图表特性：
- 支持主题切换（颜色自动适配）
- 支持图例和提示框开关
- 支持点击下钻
- 自动响应容器大小变化

## 表格组件

数据表格支持：
- 列排序（单击表头）
- 列筛选（下拉多选）
- 分页（可配置每页条数）
- 斑马纹和边框
- 行点击下钻
- 面包屑导航

## 表单组件

表单筛选器支持 6 种字段类型：
- 输入框（input）
- 下拉选择（select）
- 日期选择（date）
- 日期范围（daterange）
- 开关（switch）
- 滑块（slider）

表单联动：
- 字段值变化触发目标组件刷新
- 查询/重置按钮

## 下钻功能

### 下钻流程

1. 点击柱状图的柱子（或表格行）
2. 组件切换为下一级数据
3. 面包屑显示层级路径：`全部数据 > 华东区`
4. 点击面包屑可返回任意层级

### 下钻配置

在右侧属性面板的「交互」标签中配置：

- 触发方式：点击
- 层级定义：每一级的名称、字段名、API 接口
- 行为：替换数据
- 面包屑：显示/隐藏
- 联动组件：选择需要同步刷新的组件

## 主题系统

内置 4 套大屏主题：

| 主题 | 背景色 | 主色 |
|------|--------|------|
| 深色 | #0a1929 | #2080f0 |
| 科技蓝 | #001529 | #1890ff |
| 自然绿 | #0a1f1a | #52c41a |
| 梦幻紫 | #1a0a2e | #722ed1 |

主题影响：画布背景、组件颜色、图表配色

## 预览模式

- 5 种缩放模式：适应窗口、填充窗口、原始大小、宽度适配、高度适配
- 全屏展示（F11 或按钮）
- 底部信息栏：分辨率、缩放比例
- 控制栏自动隐藏

## 导出功能

### JSON 配置

导出完整的仪表盘配置，包含：
- 画布尺寸和主题
- 所有组件的位置、尺寸、配置
- 数据源配置
- 交互配置

### 独立 HTML

导出自包含 HTML 文件：
- 内嵌 ECharts CDN
- 原生 HTML 表格和表单
- 自适应缩放脚本
- 无需服务器即可打开
